# Lateral Flow Reader

Computer vision system for quantitative lateral flow test analysis using Raspberry Pi, Picamera2, and OpenCV for cassette detection, line tracking, and time-series signal extraction.

## Features

- Cassette quadrilateral detection
- Perspective correction
- Results window localization
- Strip ROI isolation
- Control/test line detection
- Signal strength quantification
- Time-series CSV logging
- Live visualization overlays

## Tech Stack

- Python
- OpenCV
- NumPy
- Raspberry Pi
- Picamera2
- Computer Vision
- Signal Processing
